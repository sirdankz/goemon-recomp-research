## What this repo is

A living notebook for Mystical Ninja Starring Goemon (recomp) reverse-engineering & modding:
- confirmed memory addresses (inputs, weapon slot, HP)
- confirmed hook entrypoints (DD830 / E3B9C / D98E0)
- action-code maps (`$a1`) and ctx discriminator maps (`ctx+0x4C`)
- “gotchas” (teleports, gates, sticky vs per-frame writes)
